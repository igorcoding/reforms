Наши предложения

БКИТ (3 семестр)

  • Расширить курс (больше лекций и семинаров)
  • Переход к вебу
  • Основы клиент-серверного взаимодействия:
    1. HTTP
    2. ASP.Net
    3. Базовая вёрстка
  • Практика: разработка простого веб-приложения на ASP.NET

РИП (5 семестр)

  • Продвинутое программирование на клиенте (HTML+CSS+JavaScript)
    1. CSS-препроцессоры: SASS/LESS
    2. Javascript-фреймворки: Backbone/AngularJS
    3. Семантическая вёрстка
    4. Тестирование, публикация проекта в интернете
    5. Системы сборки и оптимизации кода: Gulp/Grunt
    6. Практика: доведение проекта по БКИТ до состояния готового продукта (портфолио)

Модели данных

  • Дать современные СУБД (MySQL, MSSQL...)
  • Проектировать модели, изучать структуры запросов
  • Делать упор на SQL

ПКШ

  • Требуется курс по самому C#
  • C++ -> C#
  • Объектно-ориентированный язык для преподавания ООП
  • Добавить в курс практику применения паттернов проектирования
  • Не будет разрыва в полсеместра между C# и ASP.Net

XML -> Слабоструктурированные данные

  • Акцент на технологиях сериализации в целом
  • К XML добавляем JSON и бинарные форматы (Protobuf)
  • В конце курса рассказываем про NoSQL СУБД (вместо Exist популярная MongoDB)

АиСД (Алгоритмы и структуры данных)

  • Сложность алгоритмов
  • Простые алгоритмы и структуры данных
  • Сортировки
  • Алгоритмы на строках
  • Графы и деревья
  • Хеш-таблицы
  • Сейчас в программе нет алгоритмов, но в образовании IT-специалиста это очень важно
  • 2 семестра:
  • Простые алгоритмы + сортировки
  • Простые структуры
  • Графы
  • Сбалансированные деревья
  • RMQ, LSD, MSD
  • SparseTable
  • Хеш-таблицы
  • Алгоритмы на строках, жадные алгоритмы
